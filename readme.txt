Easily find the Least Common Multiple (LCM) of two or more numbers. Get step-by-step solutions using prime factorization, division, GCF, and Venn diagrams.

[https://www.calculator.io/lcm-calculator/](https://www.calculator.io/lcm-calculator/)

== Usage ==

Add the LCM Calculator shortcode to your page, post or sidebar:

`[calcio_lcm_calculator]`

Add the following code to your template where you would like the LCM Calculator to appear:

`<?php display_calcio_lcm_calculator(); ?>`

== Installation ==

1. Upload the LCM Calculator /calcio_lcm_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the LCM Calculator plugin through the "Plugins" menu in WordPress.
